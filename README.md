# Getting started:
## Apply on a new system
This is a alias to `task cp-hardware` and `task apply`.
```sh
task new-system
```
## Usage

### Apply

```sh
task apply
```

### Build iso
```
task iso
```
### Buid iso flash-os
```
task flash-os-iso
```
### Pull update from upstream
```sh
task pull
```
or
```sh
git pull
```
to pull and apply a update just simpily run:
```sh
task pull apply
```
