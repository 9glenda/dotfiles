> My nixos dotfiles/config deployed as a flake

### ❄️ [nixos](https://nixos.org/)

## Install

```sh
task apply
```

## Usage

### Build iso
```
cd iso
task iso
```
### Buid iso flash-os
```
cd iso
task flash-os
```
